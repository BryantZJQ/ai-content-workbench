# AI视频API集成调研

> 目标：在分镜提示词生成后，一键调用AI视频API自动生成视频片段

---

## 二、推荐方案：可灵API（阿里云百炼通道）

### 为什么选可灵
- 中文提示词理解最好，和我们生成的中文分镜提示词天然匹配
- 通过阿里云百炼调用，国内网络无障碍
- 价格合理，5秒视频约2.4-4.8元
- 文档完善，Python调用简单

### 接口流程（异步）
```
1. POST 创建任务（传入提示词）→ 返回 task_id
2. GET 轮询任务状态（用 task_id）→ 返回 video_url
3. 下载视频文件
```

### 示例成本估算
| 视频长度 | 镜头数 | 每镜头5秒 | 总成本（标准模式） |
|----------|--------|-----------|-------------------|
| 30秒短视频 | 6个 | 30秒 | ~14.4元 |
| 60秒短视频 | 10个 | 50秒 | ~24元 |
| 3分钟中视频 | 20个 | 100秒 | ~48元 |

### 接入要求
1. 注册阿里云账号
2. 开通百炼大模型服务
3. 获取 API Key
4. 在 Streamlit Secrets 中配置 `KLING_API_KEY`

## 三、集成设计方案

### 用户流程
```
分镜提示词生成完毕
    ↓
显示"🎬 一键生成AI视频"按钮（需额外付费提示）
    ↓
用户点击 → 逐个镜头调用可灵API
    ↓
显示进度：镜头1生成中... 镜头2生成中...
    ↓
全部完成 → 提供视频下载（zip包含所有片段）
    ↓
提示用户用剪映拼接
```

### 技术实现
- 新建 `video_gen.py` 模块，封装可灵API调用
- 异步轮询 + streamlit进度条
- 生成的视频暂存后提供下载
- 费用由用户自己的API Key承担（不走我们的Key）

### 卡密门控
- 视频生成作为**高级功能**，可单独收费或包含在高级卡密中
- 建议：用户自带API Key，我们不代付视频生成费用

## 四、实施优先级
