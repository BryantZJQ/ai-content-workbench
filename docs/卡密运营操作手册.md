## 二、生成卡密（你的电脑上操作）

### 打开终端

按 `Win+R`，输入 `powershell`，回车。然后输入：

```powershell
cd F:\00_tips\content-pipeline-web
```

### 生成命令

```powershell
# 生成5个试用卡（总共3次，用完即废）
python key_manager.py --plan demo --count 5

# 生成10个体验卡（3天有效期，用于引流）
python key_manager.py --plan trial --count 10

# 生成5个月卡（30天有效期，用于售卖）
python key_manager.py --plan monthly --count 5

# 生成3个年卡
python key_manager.py --plan yearly --count 3

# 查看所有已生成的卡密
python key_manager.py --list
```

## 三、把卡密配置到云端（让线上网站能验证）

### 为什么需要这一步？

卡密生成在你电脑上，保存在 `data/keys.json` 文件里。
但线上网站（Streamlit Cloud）没有这个文件，所以需要把卡密配置到 **Streamlit Secrets** 里。

### 操作步骤

1. 打开 https://share.streamlit.io
2. 找到你的应用 **ai-content-workbench**
3. 点击应用右边的 **⋮**（三个点）→ **Settings**
4. 左侧点 **Secrets**
5. 在文本框里粘贴以下格式的内容：

```toml
# DeepSeek API Key（必填，你的API密钥）
DEEPSEEK_API_KEY = "sk-8dbc7287e29248babdf27cf595d622d1"

# === 试用卡（3次用完过期）===
[keys.ACP-FSPT-MQKA-G8O9]
plan = "demo"
status = "unused"

[keys.ACP-XYYT-LX6R-HM7F]
plan = "demo"
status = "unused"

# === 体验卡（3天）===
[keys.ACP-HIAW-9B70-DITV]
plan = "trial"
status = "unused"

[keys.ACP-P1MM-7WU4-GF6B]
plan = "trial"
status = "unused"

[keys.ACP-V8VC-FS3L-PT0C]
plan = "trial"
status = "unused"

# === 月卡（30天）===
[keys.ACP-T75B-1XGF-GXCD]
plan = "monthly"
status = "unused"

[keys.ACP-FG1G-FZVC-K3PL]
plan = "monthly"
status = "unused"

[keys.ACP-SB0B-ZP5I-XEWS]
plan = "monthly"
status = "unused"
```

6. 点 **Save**，等待应用自动重启（约30秒）

### 新增卡密时

每次生成新卡密后，回到 Secrets 页面，**在已有内容后面追加**新卡密的配置块即可。

## 五、日常管理

### 查看卡密状态

```powershell
cd F:\00_tips\content-pipeline-web
python key_manager.py --list
```

输出：
```
卡密                      类型      状态      到期时间
------------------------------------------------------------
ACP-FSPT-MQKA-G8O9       试用卡    active    未激活
ACP-T75B-1XGF-GXCD       月卡      active    2026-05-03
ACP-HIAW-9B70-DITV       体验卡    expired   2026-04-06
```

### 状态说明

| 状态 | 含义 |
|------|------|
| `unused` | 未使用，还没有人激活 |
| `active` | 使用中，正常有效 |
| `expired` | 已过期（超过有效天数） |
| `exhausted` | 次数用完（试用卡专属） |

### 补货流程

当体验卡/月卡快发完时：
1. 生成新卡密：`python key_manager.py --plan monthly --count 10`
2. 把新卡密追加到 Streamlit Cloud 的 Secrets 里
3. 记录到你的卡密台账（Excel/备忘录）

## 七、常见问题

**Q: 一个卡密可以多人共用吗？**
A: 可以，但会共享额度。建议一人一码。

**Q: 卡密发错了怎么办？**
A: 直接在 `data/keys.json` 里删除对应条目，同时更新 Streamlit Secrets。

**Q: 怎么知道哪个卡密发给了谁？**
A: 建议维护一个 Excel 表格记录：卡密 → 发放对象 → 发放日期 → 类型。

**Q: 云端的试用卡用完次数后会自动提示吗？**
A: 会。用户会看到"试用次数已用完（共3次），请购买正式卡密解锁无限使用"的提示。
