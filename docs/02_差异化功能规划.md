# 差异化功能规划

> 核心问题：市面上AI脚本生成工具很多，我们的独特价值在哪里？

---

## 现状分析

### 我们已有的
| 功能 | 独特性 | 竞品覆盖度 |
|------|--------|-----------|
| 四平台热搜聚合 | ⭐⭐ | 中（单平台多，聚合少） |
| AI智能选题打分 | ⭐⭐⭐ | 低（多数工具不做选题） |
| 12风格脚本生成 | ⭐⭐ | 高（同质化严重） |
| 分镜提示词生成 | ⭐⭐⭐ | 低（多数只到脚本） |
| 一键出片流水线 | ⭐⭐⭐⭐ | 极低（选题→脚本→分镜贯通的几乎没有） |

### 竞品短板（我们的机会）
- 绝大多数工具只做 **"生成"**，不做 **"分析"**
- 没人做 **"为什么这条视频爆了"** 的逆向拆解
- 标题优化停留在"帮你起标题"，没有 **对比评分**
- 脚本写完就完了，没有 **质量诊断和优化建议**
- 内容日历/节点营销，没有工具化

---

## 差异化功能方案（按冲击力排序）

### 🔥 S级：爆款拆解器（核心杀手级功能）

**用户痛点**：看到别人的爆款视频，只会说"真厉害"，不知道为什么爆、怎么复制。

**功能**：粘贴一条爆款视频的文案/脚本，AI逆向拆解出：
- 📐 **结构公式**：开场用了什么钩子类型？正文怎么推进？结尾怎么收？
- ⚡ **爆点分析**：哪些句子是信息增量？哪些制造了情绪波动？
- 🎯 **可复制模板**：提取出可复用的"爆款公式"，一键套用到新选题

**独特价值**：从"生成内容"升级为"学会创作"，教用户钓鱼而不只是给鱼。

**实现难度**：⭐⭐（纯LLM分析，无需额外API）

---

### 🔥 A级功能

#### 1. 标题A/B测试器

**痛点**：标题决定80%的点击率，但创作者不知道哪个标题更好。

**功能**：
- 输入选题或脚本，AI生成5-10个标题变体
- 每个标题给出 **预测点击率评分**（1-10分）
- 标注评分维度：好奇心、信息量、情绪冲击、平台适配度
- 支持用户自己写标题，和AI方案对比PK

**实现难度**：⭐（纯LLM）

#### 2. 脚本质量诊断

**痛点**：脚本写完了不知道好不好，发出去才知道效果。

**功能**：
- 粘贴任意脚本（自己写的或AI生成的），AI给出诊断报告：
  - 🎣 **开场钩子强度**：前3秒能不能留住人？
  - 📈 **节奏曲线**：情绪是否有起伏？有没有"尿点"？
  - 💬 **互动潜力**：能不能引发评论？预测热门评论
  - 📊 **完播率预测**：预估观众看到哪里会划走
  - ✏️ **优化建议**：具体哪些句子要改，怎么改

**独特价值**：市面上没有"脚本医生"，这是纯增量功能。

**实现难度**：⭐⭐（纯LLM）

#### 3. 风格改写器

**痛点**：同一个选题想试不同风格，要重新生成、重新等。

**功能**：
- 已有脚本一键改写为其他风格（如：把"知识科普"改写成"震惊体"）
- 支持同时生成2-3个风格版本，横向对比
- 保留核心信息点，只改表达方式和节奏

**实现难度**：⭐（纯LLM）

---

### ⭐ B级功能

#### 4. 选题日历

**痛点**：不知道什么时候该发什么内容。

**功能**：
- 内置节日/节气/纪念日/热点预测时间表
- 根据用户赛道，提前7天推荐"踩点选题"
- 例：母亲节前5天推荐亲情类选题

**实现难度**：⭐⭐⭐（需要维护事件数据库）

#### 5. 评论区预测器

**功能**：AI预测这条脚本发出后观众会怎么评论，帮创作者：
- 提前准备置顶评论（引导讨论方向）
- 预判争议点，决定是否调整措辞

**实现难度**：⭐（纯LLM）

## 推荐实施顺序

| 优先级 | 功能 | 预计工时 | 价值 |
|--------|------|---------|------|
| 🥇 P0 | 爆款拆解器 | 2-3h | 杀手级差异化，获客利器 |
| 🥈 P1 | 标题A/B测试器 | 1-2h | 高频刚需，用户粘性 |
| 🥈 P1 | 脚本质量诊断 | 2h | 独创功能，壁垒 |
| 🥉 P2 | 风格改写器 | 1h | 提升复用率 |
| P3 | 评论区预测器 | 1h | 锦上添花 |
| P3 | 选题日历 | 3-4h | 需要数据维护 |

**核心定位转变**：
> 从"AI帮你写脚本"的工具 → **"AI教你做爆款"的创作教练**

这个定位是目前市面上没有竞品覆盖的空白地带。
